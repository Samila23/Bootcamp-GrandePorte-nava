O que é o HTML?

HTML (abreviação para a expressão inglesa HyperText Markup Language, que significa: "Linguagem de Marcação de Hipertexto" é uma linguagem de marcação utilizada na construção de páginas na Web. Documentos HTML podem ser interpretados por navegadores.

Estamos trabalhando em grupo para construção de exercícios práticos do conteúdo abordado na Aula 01 do curso de HTML.

Texto em Negrito

Podemos deixar o texto em Negrito utilizando a tag < b >

Texto em Italico

Podemos deixar o texto em Italico utilizando a tag <i>

Texto Sublinhado

Podemos sublinhar o texto utilizando a tag < u >

Texto em Sobescrito

Podemos deixar o texto em Sobrescrito utilizando a tag < sup >

Texto em Subescrito

Podemos deixar o texto em Subscrito utilizando a tag < sub >

Texto com formatação original

Podemos deixar o texto com a sua formatação original utilizando <pre>
